uni-app 是一个使用 Vue.js 开发所有前端应用的框架，开发者编写一套代码，可发布到iOS、Android、Web（响应式）、以及各种小程序（微信/支付宝/百度/头条/QQ/钉钉/淘宝）、快应用等多个平台。

# 功能

商城 （分销功能） 微信及小程序支付

实时行情

websocket

数字货币交易所 K线图

C2C

在线客服(websocket)

钱包 生成ETH,BTC,USDT（TRC20及OMNI）

资产管理 (收款，转帐，提币等)

量化机器人

会员升级购买

团队等级管理
